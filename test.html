postprocess.jsテスト用HTML

このHTMLは、postprocess.jsの挙動を確認するためのものです。

ではここで、アドレス演算子`&`を使用して、変数hoge※1のアドレスを取得してみます。

上の文章の「※1」のところをマウスオーバーすることで、脚注の内容が表示されたかと思います。

また、以下のプログラムには、行番号が付与されていると思います。

具体的な使い方はHTMLソースを参照してください。

#include <stdio.h>

int main(void)
{
    int     hoge = 5;
    int     piyo = 10;
    int     *hoge_p;

    /* それぞれの変数のアドレスを表示する */
    printf("&hoge..%p\n", (void*)&hoge);
    printf("&piyo..%p\n", (void*)&piyo);
    printf("&hoge_p..%p\n", (void*)&hoge_p);

    /* ポインタ変数hoge_pにhogeのアドレスを代入する */
    hoge_p = &hoge;
    printf("hoge_p..%p\n", (void*)hoge_p);

    /* hoge_pを経由してhogeの値を表示する */
    printf("*hoge_p..%d\n", *hoge_p);

    /* hoge_pを経由してhogeの値を変更する */
    *hoge_p = 10;
    printf("hoge..%d\n", hoge);

    return 0;
}

このプログラムは、拙著「C言語ポインタ完全制覇」(改訂版)のList1-4 pointer.c(p.44)からの抜粋です。

以下の空行は、※1のクリックで脚注にジャンプできることを示すためのものです。
